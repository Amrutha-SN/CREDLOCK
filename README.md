# CREDLOCK
Credlock is a secure desktop password manager built with Python and CustomTkinter.The primary aim of Credlock is to provide users with a secure, 
convenient, and feature-rich desktop tool for managing sensitive credentials. It offers:

- Password strength evaluation using the zxcvbn library
- Breach detection via the Have I Been Pwned (HIBP) API
- A built-in password generator for creating strong, random credentials
- Organized storage for Passkeys, WiFi passwords, and secret Codes
- A master PIN login to protect the access to the app

The design focuses on a graphical user interface (GUI) built with customtkinter to ensure a positive user experience while prioritizing data security and easy access to stored information.
